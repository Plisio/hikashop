# Plisio Payment Gateway for Hikashop
This plugin allows you to use Plisio as a payment gateway to process payments for your shop based on Hikashop platform.
## Installation
### Install via Add-ons manager
1. Download the latest plugin from the [release page](https://github.com/Plisio/hikashop/releases).
2. Go to **Admin Panel** > **System** > **Extensions** > **Upload plugin zip file** > **Manage Extensions** > **Find Plisio Payment Gateway plugin** > **Enable the plugin**.
3. Go to **Admin Panel** > **Components** > **HikaShop** > **Configuration** > **System** > **Payment Methods** > **New** > **Click on Plisio Payment Gateway plugin** > **Enter your Secret Key** > **Save settings**.
